Кто мы?

Команда из двух программистов которая любит программировать чат-ботов и разные движки

Из кого мы состоим?

  • Cha14ka
  • Lanode

Что мы делаем?

  • AugВК - нативный легковесный мессенджер Вконтакте (WIP)
  • AugWorkstation - простой серверный Docker дистрибьютив с программой-клиентом для создания мультипользовательского удаленного рабочего места
  • KBot - серия чат-ботов. Репозитории с ними вы сможете найти у нас на GitHub. Страница бота ВКонтакте
  • AugOCR - программа для перевода текста прямо на экране компьютера (WIP)
  • MygryBot (бывш. MerkelBot) - Бот для привествий в беседе группы и исполнения прикольных команд сделанный специально для компании Мыгры.
  • AugVNEngine - Движок для создания визуальных новелл с собственным скриптовым языком и поддержкой расширений сцен через Godot.
  • Trinity Bot - Бот работающий с 2018 года для парсинга форума Trinity-GTA и постинга новостей фракций в предложку официальной группы новостей сервера.
